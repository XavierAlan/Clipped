# Clipped

Clipped is a small daily web game where you guess a music video from a sequence of still frames, inspired by Framed.wtf but focused on music videos.

## Overview

- One music video per day for all players.
- Frames are revealed progressively after each wrong guess.
- Limited number of attempts to find the correct video.

## Tech stack (planned)

- Backend: Node.js / Next.js API routes
- Deployment: Vercel
- Media preprocessing: FFmpeg (offline frame extraction)
